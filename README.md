# Simple Calculator with C++
Do your basic mathmatics operations with this program.
Later, it will receive updates so that you can use more operations such as power or square. Enjoy it!

You just have to follow steps, nothing difficult.

## Dependencies

You need these packages in you computer to compile the project: **git, g++**

## Build
You can run these commands to use it :

> git clone git@github.com:MitsoaAllan/simple-calculator.git
>
> cd simple-calculator
>
> g++ main.cpp -o main
>
>./main
